# Telecom Customer Churn Dashboard

## Project Overview

This project presents an interactive Power BI dashboard for analyzing customer churn in the telecom industry. The dashboard helps identify customer retention patterns and provides business insights for decision-making.

## Tools Used

- Power BI
- Microsoft Excel
- DAX
- Data Visualization

## Dashboard Features

- Customer Churn Analysis
- Customer Retention Insights
- Contract Type Analysis
- Monthly Charges Analysis
- Payment Method Analysis
- Interactive Filters and Slicers
